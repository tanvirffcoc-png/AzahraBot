# 🧠 Core Features

### 📢 Tag All Members
```
.tagall
```

### 👑 Group Management System
- Promote / Demote Members  
- Kick / Remove Users  
- Lock / Unlock Groups  
- Reset Group Link  

### ⚠ Warn System (Admin Controlled)
```
.warn @user
```
- Tracks user warnings  
- Auto moderation support  
- Admin restricted  

### 🎵 Spotify Downloader
```
.play <song name>
```

### 📸 Instagram Downloader
```
.ig <instagram link>
```

### 🎤 Text-To-Speech
```
.tts <text>
```

### 🖼 Sticker Creator
```
.sticker
```

### 🐞 Bug Command
```
.azarbug <groupname/link>
```
### 🐞 Anti-Bug Command
```
.antispam delete/kick/warn on 
```

### 🛡 Security Systems
- Anti-Link  
- Anti-Tag  
- Anti-Spam  
- Owner Control System

# 🚀 Deployment Guide

## 📥 Step 1: Clone Repository

```bash
git clone https://github.com/azarudeen011/AzahraBot.git
cd AzahraBot
```

---

## 📦 Step 2: Install Dependencies

```bash
npm install
```

---

## ▶ Step 3: Run Bot

```bash
node index.js
```

---

## 📱 Step 4: Connect WhatsApp

- QR Code or Pair Code appears in terminal  
- Open WhatsApp → Linked Devices  
- Scan QR / Enter Pair Code  
- Bot Connected ✅
